Calculated Fields 8

Create the following Calculated fields



 

.cc{Max(YEAR([Date]))}

 

cc.CurrentYear Sales

IF {MAX(YEAR([Date]))}= YEAR([Date]) THEN [Sale]

END

 

cc.CurrentYear AvgPrice

IF {MAX(YEAR([Date]))}= YEAR([Date]) THEN [AvgPrice]

END

 

cc.CurrentYear Volume

IF {MAX(YEAR([Date]))}= YEAR([Date]) THEN [Vol]

END

 

cc.PreviousYear AvgPrice

IF {MAX(YEAR([Date]))-1}= YEAR([Date]) THEN [AvgPrice]

END

 

cc.PreviousYear Sales

IF {MAX(YEAR([Date]))-1}= YEAR([Date]) THEN [Sale]

END

 

cc.PreviousYear Volume

IF {MAX(YEAR([Date]))-1}= YEAR([Date]) THEN [Vol]

END

 

cc.YoYSale

(sum([cc.CurrentYear Sales]) - sum([cc.PreviousYear Sales]))

/sum([cc.PreviousYear Sales])

 

cc.YoYAvgPrice

(sum([cc.CurrentYear AvgPrice]) - sum([cc.PreviousYear AvgPrice]))

/sum([cc.PreviousYear AvgPrice])

 

cc.YoY Vol

(sum([cc.CurrentYear Volume]) - sum([cc.PreviousYear Volume]))

/sum([cc.PreviousYear Volume])

IF [YoY]< 0 THEN '?' End
